**Gestione Utenti:**
- `GET /utenti` - Recupera tutti gli utenti dal database  
- `POST /utenti` - Inserisce nuovo utente (con `data_registrazione` automatica)

- `POST /change-password` - Cambio password con validazione sicurezza
